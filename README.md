# 🚖 Taxi Go – iOS Ride Booking App

Taxi Go is a modern iOS ride-booking application built using **SwiftUI** and **Firebase Authentication**, designed to provide a seamless experience for users to book rides and manage profiles.

## ✨ Features

- 🔐 Firebase Authentication (Login & Signup)
- 🚗 Ride Booking UI
- 🧭 Multiple Services (Taxi, Rental, Intercity, Package)
- 👤 User Profile & Stats
- 🧾 Clean SwiftUI Interface
